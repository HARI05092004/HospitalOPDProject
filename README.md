# HospitalOPDProject
## Hospital OPD Online Appointment System

This project allows patients to book OPD appointments and enables admins to manage OPD bookings by accepting or rejecting appointments.

### Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express.js
- Database: MongoDB

### Features
- Patient appointment booking
- Admin login
- View OPD appointments
- Accept / Reject appointments
- Appointment status stored in MongoDB
